# Code of Conduct

[English](./CODE_OF_CONDUCT.md) | [简体中文](./CODE_OF_CONDUCT.zh-CN.md)

## 我们的标准

我们希望这个仓库对贡献者和使用者来说都是有用、专业且安全的。

有助于营造积极协作环境的行为包括：

- 在技术分歧中保持尊重
- 先假设对方出于善意，同时保持表达准确
- 提供可执行的 review 反馈
- 通过说明权衡来解决问题，而不是升级冲突

不可接受的行为包括：

- 骚扰、辱骂或恐吓
- 人身攻击
- 泄露他人隐私信息
- 故意制造混乱或以恶意方式参与协作

## 适用范围

本行为准则适用于仓库的 issue、pull request、discussion 以及其他由项目管理的协作空间。

## 执行

项目维护者可以删除、编辑或拒绝违反本行为准则的评论、提交、代码、issue 及其他贡献内容。

## 举报

如果遇到行为问题，请优先通过维护者的 GitHub 主页或其他私下渠道联系维护者，再决定是否公开升级处理。
