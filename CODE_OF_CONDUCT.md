# Code of Conduct

[English](./CODE_OF_CONDUCT.md) | [简体中文](./CODE_OF_CONDUCT.zh-CN.md)

## Our Standards

We want this repository to be useful, professional, and safe for contributors and users.

Examples of behavior that contributes to a positive environment include:

- being respectful in technical disagreement
- assuming good intent while still being precise
- giving actionable review feedback
- documenting tradeoffs instead of escalating conflict

Examples of unacceptable behavior include:

- harassment, abuse, or intimidation
- personal attacks
- doxxing or publishing private information
- intentionally disruptive or bad-faith participation

## Scope

This Code of Conduct applies in repository issues, pull requests, discussions, and other project-managed collaboration spaces.

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, issues, and other contributions that violate this Code of Conduct.

## Reporting

For conduct problems, contact the repository maintainer privately through the maintainer's GitHub profile or other private channel before escalating publicly.
