# Knowledge Base Boundary

[English](./README.md) | [简体中文](./README.zh-CN.md)

`knowledge-base/` 包含提取后的 Agiso 文档材料以及用于再生成这些材料的流水线。
使用方应将该目录视为本地证据库，而不是手写源文档。

## 来源与再分发

- `docs/`、`extracted/` 和 `raw_js/` 是与 Agiso 公共属性相关的第三方材料或派生材料。
- `extract_docs.py` 是用于转换或索引这些材料的仓库代码。
- 仓库的 MIT License 并不会自动赋予你对本目录第三方材料的再分发权。

如果你计划公开发布本仓库，请先阅读 `../THIRD_PARTY_NOTICES.md`。
