# Support

## How To Get Help

- Use GitHub Issues for bugs, gaps in repository documentation, or improvement requests.
- Use pull requests for concrete fixes with clear scope.

## Good Support Requests

Include:

- what you were trying to do
- which repository area is involved
- the exact file or script path when relevant
- the observed result and expected result

## Not In Scope

- requests for access to private Agiso accounts or credentials
- legal interpretation of third-party documentation rights
- support for unpublished internal systems that are not represented in this repository
