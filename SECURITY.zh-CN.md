# Security Policy

[English](./SECURITY.md) | [简体中文](./SECURITY.zh-CN.md)

## 范围

本仓库主要包含文档资产、提取工具以及一个 Codex skill。

## 报告漏洞

如果你怀疑存在可能暴露密钥、凭据或私有基础设施细节的安全问题，不要直接创建公开 issue。

请改为：

1. 通过 GitHub 私下联系维护者
2. 提供足够的复现信息
3. 在公开披露前给予合理的分诊时间

## 敏感数据规则

- 不要提交令牌、Cookie、密码或 API 密钥
- 不要提交私有商家或客户数据
- 发布示例前先完成脱敏
